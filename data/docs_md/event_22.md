                ## Description Complète
                <p><em>One Minute Sculptures (Sculptures d'une minute),</em> 1996-2017<br>Depuis 1996, Erwin Wurm développe le concept interactif des <em>One Minute Sculptures (Sculptures d'une minute)</em>, qui propose au visiteur de devenir lui-même une sculpture en réalisant les consignes données par l'artiste. Dans les <em>One Minute Sculptures</em>, des objets pris dans un environnement immédiat et des “modèles” sont associés sans hiérarchie les uns aux autres pour former autant de sculptures provisoires. Le propos est de renouveler fondamentalement la sculpture, porteuse d’infinies potentialités.<br>«&nbsp;Je m'intéresse à la vie de tous les jours. Tous les matériaux qui m'entourent peuvent être utilisés, aussi bien que les objets, les sujets impliqués dans la société contemporaine. Mon travail parle de l'être humain dans toutes ses dimensions : physique, spirituelle, psychologique et politique&nbsp;»</p>
